# VectorShift Flow Builder

An interactive flow builder built using React and React Flow that allows users to create, connect, and manage custom nodes dynamically.

---

## 🚀 Features

- Create and connect nodes visually
- Reusable BaseNode architecture (clean component design)
- Dynamic input handling
- Scalable node structure for future extensions

---

## 🧠 Key Highlight

Implemented a reusable `BaseNode` component to eliminate duplication across different node types, improving maintainability and scalability.

---

## 🛠 Tech Stack

- React
- React Flow
- JavaScript
- Tailwind CSS

---

## 📂 Project Structure

frontend/ → React application  
backend/ → API services
